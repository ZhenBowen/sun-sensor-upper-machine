# 太阳敏上位机 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建第一版可运行的太阳敏工程测试型上位机，支持模拟数据、推荐二进制遥测协议解析、实时显示、曲线、CSV 记录和真实串口预留。

**Architecture:** 采用分层结构：`sun_protocol.py` 负责帧格式和 CRC，`sun_models.py` 定义数据结构，`sun_simulator.py` 生成模拟遥测，`sun_host.py` 管理串口/模拟数据源，`sun_logger.py` 管理 CSV，`sun_monitor.py` 和 `sun_gui.py` 负责 PyQt5 + pyqtgraph 界面。第一版不依赖 STM32 固件即可运行，真实硬件出来后只需要按 ICD 调整协议层。

**Tech Stack:** Python 3.11, PyQt5, pyqtgraph, pyserial, unittest, csv.

---

## File Structure

- Create: `sun_models.py`，遥测、统计、记录上下文数据类。
- Create: `sun_protocol.py`，CRC-16/Modbus、32 字节遥测帧打包/解析、遥控命令打包、字节流解析器。
- Create: `sun_simulator.py`，10 Hz 模拟太阳敏数据源，支持正常、CRC 错误、丢帧模式。
- Create: `sun_logger.py`，稳定 CSV 表头和逐帧记录。
- Create: `sun_host.py`，Qt 线程封装，统一模拟模式和真实串口模式。
- Create: `sun_monitor.py`，实时数值表、状态位解释、角度/ADC/温度曲线。
- Create: `sun_gui.py`，主窗口、连接控制、记录控制、标定输入、命令发送区。
- Create: `run_sun_gui.py`，程序入口。
- Create: `tests/test_sun_protocol.py`，协议单元测试。
- Create: `tests/test_sun_logger.py`，CSV 单元测试。
- Create: `tests/run_tests.py`，无需 pytest 的 unittest 入口。
- Create: `logs/.gitkeep`，默认日志目录占位。
